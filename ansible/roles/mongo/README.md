# mongo #

A role for installing and configuring MongoDB servers.

## Requirements ##

None

## Role Variables ##

None

## Dependencies ##

None

## Example Playbook ##

Here's how to use it in a playbook:

```yaml
- hosts: mongo
  become: yes
  become_method: ansible.builtin.sudo
  roles:
     - mongo
```
